# MyShop

## install

### requirements
  In order to build the application the following tools required:
  node.js, npm, bower

### build
  run the following commands in the root of the project:

 * npm install
 * bower install
 * gulp build
 * open public/index.html


## test framework
  Mocha is used as test framework

### execute test
  * open test/index.html
